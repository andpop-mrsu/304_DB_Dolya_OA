Лабораторная работа 8: CRUD-приложение для учёта студентов и экзаменов
Описание

Веб-приложение для управления составом учебных групп и результатами сдачи экзаменов с полным CRUD-функционалом: список студентов, фильтрация по группе, редактирование персональных данных, просмотр и ввод оценок (включая ввод «задним числом» за любой учебный год).
Структура проекта

Task08/
├── data/
│ ├── db.db
│ └── db.sqlite # База данных SQLite
├── public/
│ ├── index.php # Список студентов
│ ├── edit.php # Добавление/редактирование студента
│ ├── delete.php # Удаление студента
│ ├── exam_results.php # Результаты экзаменов студента
│ ├── edit_exam.php # Редактирование оценки
│ ├── delete_exam.php # Удаление оценки
│ └── add_exam.php # Добавление оценки (задним числом)
├── db_init.sql # SQL-скрипт инициализации БД
└── README.md # Этот файл

Установка и запуск

    1. Запуск веб-сервера: php -S localhost:3000 -t public
    2. Открыть в браузере: http://localhost:3000/
